Hardware Setup

Wiring the OLED to ESP32

Only 4 wires needed! The OLED uses I2C protocol.

OLED Pin ESP32 Pin Description
VCC 3.3V Power — use 3.3V, NOT 5V
GND GND Ground
SDA GPIO 21 Data line
SCL GPIO 22 Clock line
⚠️ Use 3.3V, not 5V! The ESP32 is a 3.3V board. Connecting OLED VCC to 5V might damage it.
Setup Guide

Step-by-Step Instructions

1

Install Arduino IDE & ESP32 Support

Download Arduino IDE from arduino.cc/en/software (free). Then add ESP32 board support:

Go to File → Preferences, and in "Additional Boards Manager URLs" paste:

https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json

Then go to Tools → Board → Boards Manager, search esp32, and install "esp32 by Espressif Systems".

2

Install Required Libraries

In Arduino IDE, go to Tools → Manage Libraries and install these three:

  • Adafruit SSD1306 — OLED driver
  • Adafruit GFX Library — Graphics engine (usually auto-installs)
  • ArduinoJson — JSON parsing
3

Configure the Firmware

Open esp32_firmware/esp32_lyrics_display.ino in Arduino IDE. Edit these 3 lines at the top:

const char* WIFI_SSID     = "YourWiFiName";
const char* WIFI_PASSWORD = "YourWiFiPassword";
const char* PC_IP_ADDRESS = "192.168.1.XXX";  // ← your PC's IP

To find your PC's IP: Windows: run ipconfig in Command Prompt → look for "IPv4 Address". Mac/Linux: run ifconfig or check WiFi settings.

4

Upload to ESP32

Connect your ESP32 via USB. In Arduino IDE:

  • Select Tools → Board → ESP32 Arduino → ESP32 Dev Module
  • Select Tools → Port → COMx (Windows) or /dev/ttyUSB0 (Linux/Mac)
  • Click the Upload (→) button
💡If upload fails, hold the BOOT button on the ESP32 while clicking Upload, then release when "Connecting..." appears.
5

Install Python & Run the PC Bridge

Download Python from python.org (3.8 or newer). Make sure to check "Add Python to PATH" during install.

Then double-click start_bridge_windows.bat (Windows) or run in terminal:

cd pc_bridge
pip install -r requirements.txt
python pc_bridge.py

You should see: Server running at http://0.0.0.0:5000

🔍Open http://localhost:5000/now-playing in your browser to check if it's working before powering the ESP32.
6

Power On & Enjoy!

Plug the ESP32 into any USB power source (phone charger, power bank, PC USB port). It will:

  • Connect to your WiFi automatically
  • Show the clock screen by default
  • Automatically show lyrics when you play music on Spotify or YouTube Music
🎉The ESP32 and PC must be on the same WiFi network. The PC bridge must be running whenever you want lyrics.

Help

Troubleshooting

🖥 OLED shows nothing / black screen
Check your wiring. Try changing SCREEN_ADDRESS from 0x3C to 0x3D in the .ino file.
📡 ESP32 can't connect to WiFi
Double-check SSID and password. Make sure it's 2.4GHz (ESP32 doesn't support 5GHz WiFi).
🎵 No lyrics showing (just clock)
Make sure the PC bridge is running. Test http://localhost:5000/now-playing in a browser. Check that PC_IP_ADDRESS in the .ino matches your PC's actual local IP.
🔥 Upload fails ("A fatal error occurred")
Hold the BOOT button on the ESP32 when uploading. Or try a different USB cable (some are charge-only).
🎵 Song detected but no lyrics
Some songs don't have lyrics in the lrclib.net database. Try a more popular song to test. The song title & artist must match exactly.
